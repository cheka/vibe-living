# Security Policy

## Supported versions

Security fixes are applied to the latest release on the default branch.

## Reporting a vulnerability

Please use GitHub's private vulnerability reporting feature instead of opening a public issue. Include reproduction steps, affected versions, and the potential impact. Do not include unrelated personal data, prompts, or repository content.

Vibe Living is designed to run locally and without network access. Reports involving unexpected file access, command execution, hook interference, or data transmission are treated as high priority.
