# Changelog

All notable changes are documented here. This project follows semantic versioning.

## [Unreleased]

### Added

- Added a public Claude plugin marketplace catalog and persistent installation instructions for macOS Claude Desktop Local Code sessions and Claude Code CLI.

### Fixed

- Refined seated-twist motion around a fixed body axis with a clear, naturally bent-arm silhouette, and replaced the wrist exercise's synchronized hand bounce with subtle wrist-and-finger articulation while preserving the existing pixel figure and overlay style.

## [0.2.0] - 2026-07-18

### Added

- Added automatic Simplified Chinese and English overlay localization based on the macOS preferred language.
- Added deterministic Chinese and English previews for movement and hydration prompts.
- Added a dependency-free, headless lifecycle Harness for local development and CI.
- Added versioned product specifications and a mandatory spec-first development workflow.
- Added a quiet, desk-friendly hydration reminder.
- GitHub-ready marketplace repository structure.
- Codex and Claude Code manifests.
- Native macOS pixel movement companion.
- Lifecycle-aware show, hide, pause, and multi-session behavior.
- English and Simplified Chinese documentation.
- Unit tests, validation scripts, and macOS CI.

### Changed

- Constrained the movement catalog to quiet, low-amplitude actions within a personal desk space.
- Replaced the large overhead reach and side bend with wrist relaxation and a quiet posture reset.

## [0.1.0] - 2026-07-18

- Initial public-development release.
